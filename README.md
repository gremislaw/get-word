# Words API

Простой API для получения случайных русских слов 4|5|6 длины.

## Файлы

- `words_russian.json` - основной файл с русскиси словами (4, 5 и 6 букв)
- `words_english.json` - основной файл с английскиси словами (4, 5 и 6 букв)
- `index.html` - веб-страница для GitHub Pages

## Использование API

### Получить случайное слово

```http
GET https://your-username.github.io/your-repo/word?length=5
```

### Параметры:

- length - количество букв (4, 5 или 6). По умолчанию 5.

### Ответ:

```json
{
  "word": "СЛОВО"
}
```

### Количество слов
4 буквы: ~1500 слов

5 букв: ~3300 слов

6 букв: ~4600 слов
